It is a plugin for importing and exporting Term, Post information.

== Description ==

Custom Importer & Exporter is a plug-in for exporting and importing post information and custom taxonomy term division.
Data is exported in Json format.

== Installation ==

1. Upload the 'custom-importer-exporter' folder to the / wp-content / plugins / directory.
2. Activate the Custom Importer & Exporter plug-in from the plug-in menu.
